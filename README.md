# Robtaylor Openvaf

## How do I install these formulae?

`brew install robtaylor/openvaf/<formula>`

Or `brew tap robtaylor/openvaf` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "robtaylor/openvaf"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
